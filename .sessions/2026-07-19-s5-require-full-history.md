## What shipped

Added `require_full_history(run) -> HistoryVerdict` (+ a frozen `HistoryVerdict` dataclass and `FULL`/`SHALLOW`/`UNKNOWN` constants) to `scripts/_git_truth.py` — the single home of the "a shallow (grafted) clone silently truncates history, so any git-log-derived count is unreliable → refuse/degrade" decision the fleet-measurement passes kept re-deriving independently. It is built directly on the existing `is_shallow` primitive, so the shallow-probe policy stays in one place: provably-not-shallow → `FULL` (empty reason); provably-shallow → `SHALLOW` (carrying a paste-ready `git fetch --unshallow` hint); undeterminable → `UNKNOWN` (git unavailable — "couldn't ask" is deliberately NOT read as "provably full", mirroring `is_shallow` returning `None`).

Then reimplemented `scripts/measure_grounded_skills.py._is_shallow(repo_dir)` — previously a duplicate `rev-parse --is-shallow-repository` subprocess — as a thin adapter over the shared helper (`require_full_history(make_runner(repo_dir)).verdict == SHALLOW`), deleting the second copy of the shallow-detection logic. Behavior-preserving on every axis: shallow→True, full→False, git-unavailable→False (UNKNOWN maps to not-shallow, preserving the R3 `--json` REFUSE guard's original fail-open-to-proceed). The module's `_is_shallow` name is kept as the (now thin) seam the existing refuse/writes tests monkeypatch, so `test_measure_grounded_skills.py` is unchanged.

Repo-tooling only: `scripts/_git_truth.py` never ships in `dist/bootstrap.py`, and the change does NOT touch the three parity-pinned primitives (`make_runner`/`is_shallow`/`provable_ancestry`, `AncestryAnswer`) or the dist-shipped engine port `src/engine/lib/git_truth.py` — so `dist/bootstrap.py` rebuilds byte-identical (empty diff) and `test_git_truth.py::TestEngineParity` stays green.

Files: `scripts/_git_truth.py` (new helper), `scripts/measure_grounded_skills.py` (`_git_truth` import + `_is_shallow` adapter), `tests/test_git_truth.py` (+6 tests: FULL/SHALLOW/UNKNOWN via fake runners, plus a real full-clone verdict and a real `--depth 1` shallow-clone verdict).

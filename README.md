# substrate-kit

The portable **substrate** for AI-run repositories — the workflow, orientation, governance, and
tooling layer (constitution, collaboration model, session journal, orientation router, hooks, and
checkers) extracted from the `superbot` program so any repository can adopt it.

`superbot-next` is the first consumer: it adopts the kit from here rather than inheriting it by
cloning an existing repo.

**Status:** bootstrapping — intent-only first commit; kit content ports in from superbot's proven
machinery.
